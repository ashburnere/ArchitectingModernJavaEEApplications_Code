Code examples from chapters 2, 3 and 4 for various JAVA EE features (one project for each).

cdi_events
- a CDI event is emmitted when a car is created
- usage:
	- create car with manufacturer "BMW": POST http://localhost:8080/cdi_events/resources/cars/BMW 
	- get all cars: GET http://localhost:8080/cdi_events/resources/cars
	- get car with ID "1": GET http://localhost:8080/cdi_events/resources/cars/1 - 

- ...
